硕士在读(工具篇)

发表于 2021-04-09 更新于 2021-06-09

1. Blog搭建分享

创建一个自定义博客，既方便于知识的积累，又可以加以分享。对于个人自己搭建一个博客框架来说可能成本较高，但是也是有免费的方式的。本人采用gitee page(https://gitee.com/)搭建免费的个人静态博客，当然我更倾心于github page的方式，但是奈何由于国外的github page访问速度慢且不稳定，转用国内开源代码托管平台gitee(码云)。

个人也是根据网上的教程一步步搭建了一个初步的博客模型，我选择使用的是gitee和hexo进行搭建，对博客并未进行深层次的美化，有兴趣的小伙伴可以自行研究啊。下面就push几个教程给大家。

基于 Gitee+Hexo 搭建个人博客
Hexo-GitHub 搭建个人博客配置教程
Hexo 搭建个人博客系列：主题美化篇

跟着教程走就能搭建出属于自己的Blog啦！！！我写文章的markdown语法工具是Typora。
当你搭建好自己的博客的时候，这时候必要的事图片的引用，这是我们需要结合gitee仓库和PicGo打造自己的图床，mac 系统选择.dmg下载，windwos 选择.exe 系统。

2. 英文文献搜索与下载

读文献是科研的第一步，中文文献在读的学生都能搞定，下面安利一下英文文献的下载方法。

《大木虫学术导航》，这是一个宝藏网站，类似一个科研文献资源库，自行探索。

我自己常用的就是将《熊猫学术》和《SCI-HUB》联用，对于95%的英文文献都可以轻松解决。以”machine learning”为关键词进行检索，部分右侧点击可以获取PDF版文献，部分没有的话我们点击进去搜寻文章的DOI号并利用《SCI-HUB》进行下载。

点击对应文章下面的的 ”号就可以获取对文献的引用格式了。

2. 访问谷歌

不吹不黑，谷歌搜索引擎天下第一，搜索效率奇高，好的搜索引擎真的是很省搜索资源的时间。

1. 《谷歌访问助手》，百度云提取码65gw。需要开启浏览器的的拓展程序，解压拖进去crx文件就好。时好时坏，有时候不好使。

2. 《谷歌学术镜像》，里面有五个镜像搜索，有时不稳定，多尝试尝试，总会有好用的。

3. 便笺

Windows10的便笺真的是很好用的一个小笔记便签，可以安排一下日计划、周计划、月计划、年计划，做一些小笔记都是极好的。

4. PPT模板

《优品PPT》，研究生必不可少的组会汇报，必不可少的PPT制作，优品PPT提供海量免费PPT模板。

5. 文献管理工具Zotero

《Zotero》，很好用的文献管理工具，配合《chrome应用商店》中的Zotero Connector插件一起使用。

下载好Zotero Connector浏览器插件，再配和Zotero的设置，我们就能轻松的管理文献了。

当然，当你能够访问chrome应用商店，你肯定也会发现许多有趣又实用的浏览器小插件。

6. 公式识别神器mathpix snipping tool

《mathpix snipping tool》，如果平时的科研工具需要输入大量的公式，这个公式识别配合上mathtype就能轻松完成公式的输入。
